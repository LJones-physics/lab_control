ALP ImageConverter Sample:

The intention is to show ALP users an example how to convert an ordinary
image file (.png/.bmp/.jpg/.jpeg) into the two most common ALP API formats
(ALP_DATA_MSB_ALIGN or ALP_DATA_BINARY_TOPDOWN).
Additional notes on the data formats can be found in the AlpSeqPut chapter
of the ALP-4 API description.

The program also offers the ability to scale the frames while keeping the
aspect ratio. This might be useful when using the AOI function of the
ALP API.
